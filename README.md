## ‚ú® Features
- **Real-time Data**: Fetches authentic user data from the [JSONPlaceholder API](https://jsonplaceholder.typicode.com/).
- **Dynamic Filtering**: Instantly filter donors by **Blood Group** or search for them by **City**.
- **Responsive Grid**: A sleek, dark-themed dashboard that adapts from a 4-column layout on desktops to a single column on mobile devices.
- **Smart Data Mapping**: Uses JavaScript's spread operator to inject custom properties (like Blood Groups) into existing API data.
